# V. Words and Expressions Commonly Misused

Common errors in word choice and usage. When in doubt about a specific word, check here first.

---

## Quick-Reference Table

| Word/Phrase | Problem | Fix |
|-------------|---------|-----|
| **all right** | One word form (*alright*) is non-standard | Write as two words: *all right* |
| **as good or better than** | Elliptical construction breaks grammar | "as good as his, or better" |
| **as to whether** | Padded | *whether* alone is sufficient |
| **but** (after *doubt* or *help*) | Redundant | "no doubt that" not "no doubt but that" |
| **can** vs **may** | *Can* = ability. *May* = permission | Don't use *can* when you mean *may* |
| **case** | Usually unnecessary filler | "Many rooms were poorly ventilated" not "In many cases, rooms were poorly ventilated" |
| **certainly** | Hollow intensifier | Remove or replace with a specific qualifier |
| **character** | Often redundant | "hostile acts" not "acts of a hostile character" |
| **claim** (as substitute for *say*) | *Claim* implies dispute | Use *say*, *state*, *report*, or *argue* unless disputing |
| **clever** | Overused; now means little | Use specific: *inventive*, *efficient*, *elegant* |
| **compare to** vs **compare with** | *To*: point out resemblance between unlike things. *With*: examine differences between like things | "Life is compared to a journey." "This PR was compared with the previous one." |
| **consider** (+ as) | *Consider* meaning "believe to be" takes no *as* | "I consider it complete" not "I consider it as complete" |
| **data** | Plural noun | "These data show" not "This data shows" |
| **dependable** | Unnecessary substitute | Use *reliable* or *trustworthy* |
| **different than** | Non-standard in most contexts | Use *different from* |
| **divided into** vs **composed of** | *Divided into*: split from a whole. *Composed of*: made up of parts | "The module is composed of three classes." "The test suite is divided into unit and integration tests." |
| **due to** (adverbial) | *Due to* modifies nouns, not verbs | "He failed because of a misconfiguration" not "He failed due to a misconfiguration" |
| **effect** vs **affect** | *Effect* (noun) = result. *Affect* (verb) = to influence | "The change affected performance." "The effect was a 20% speedup." |
| **etc.** | Vague when readers need specifics | Only use after a list already given in full, or when remaining items are immaterial |
| **fact** | Use only for directly verifiable statements | "The deploy completed at 14:32" is a fact. "The refactor was successful" is a judgment. |
| **factor** | Hackneyed | Rewrite: "He won by being better trained" not "Training was a key factor in his win" |
| **feature** (verb, advertising sense) | Jargon | Say what the thing does |
| **folk** | Collective singular | "The folk were..." not "The folks were..." |
| **however** | Parenthetic; needs commas | "This approach, however, introduces latency." Never open a clause with *however* joined by a comma alone. |
| **interesting** | Hollow — everyone finds different things interesting | Say what makes it interesting, or cut |
| **kind of / sort of** | Colloquial hedging | Commit to the statement or qualify specifically |
| **less** vs **fewer** | *Fewer* for countable items, *less* for quantities | "fewer requests," "less traffic" |
| **like** (as conjunction) | Non-standard in formal writing | "as I did" not "like I did" |
| **literally** | Overused as intensifier | Use only when you mean the literal meaning |
| **one of the most** | Near-meaningless superlative | State what's specific about it, or cut |
| **people** vs **persons** | *People* is standard for groups. *Persons* is formal/legal | Use *people* in most contexts |
| **split infinitive** | Widely defended; avoid when it reads awkwardly | "to quickly deploy" is usually fine; revise when it clunks |
| **that** vs **which** | *That* introduces restrictive clauses (no comma). *Which* introduces non-restrictive clauses (with comma). | "The file that contains the config" vs "The config file, which was updated last week," |
| **utilize** | Almost always just *use* | Use *use* |
| **very** | Weakens rather than strengthens | Cut or replace with a specific qualifier |
| **while** (= although/whereas) | Can cause ambiguity with its temporal meaning | Use *although* or *whereas* when not describing simultaneous action |

## On Overworked Words

Some words are so common that they've lost force. When you reach for them, stop and ask: what do I actually mean?

**Interesting** — What's interesting about it? Say that instead.  
**Certainly / definitely / clearly** — If it's clear, state it plainly. The intensifier adds nothing.  
**Very / really / truly** — These dilute rather than amplify. Cut them and state the thing directly.  
**Important / significant / notable** — Specify why it matters, or let the fact speak.
