# Federal-State Research Playbook

Use this playbook to run consistent federal and state scans for marketing compliance intelligence.

## Federal Baseline Pass

1. Identify federal control themes in scope for the campaign context.
2. Extract required/prohibited claim behaviors.
3. Capture disclosure and substantiation expectations.
4. Tag controls by channel and product type.

## State Delta Pass

1. Select target states by distribution footprint.
2. Gather state-level bulletins, circulars, and DOI guidance.
3. Compare state positions to federal baseline.
4. Record stricter-state requirements as override controls.
5. Mark unresolved conflicts for escalation.

## Delta Classification

| Delta type | Description | Action |
|---|---|---|
| Additive | State adds requirements beyond baseline | Add state-specific guardrail |
| Restrictive | State narrows allowable claims/channels | Apply stricter copy and approval routing |
| Procedural | Different filing/approval timing or format expectations | Update workflow timing and ownership |
| Ambiguous | Guidance conflicts or lacks clarity | Escalate for legal interpretation |
