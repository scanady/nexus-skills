# 3P Update Template

Use for recurring team updates where leaders need fast status visibility.

## Required Inputs

- Reporting period
- Progress since last update
- Plan for next period
- Problems/blockers
- Support needed

## Structure

1. Period and scope
2. Progress
3. Plan
4. Problems
5. Asks / dependencies

## Writing Rules

- Keep each section short and evidence-based.
- Use bullet points where possible.
- Include owner and due date for unresolved blockers.
