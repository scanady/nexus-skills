# Set Critique Rubric

Use this reference when reviewing existing imagery or generated outputs as a family. Individual image quality is not enough; the set must work as a system.

## Review Passes

### 1. Placement Fit

Check whether each image works where it will appear.

- Correct aspect ratio or enough room to crop
- Focal point survives desktop and mobile crops
- Text-safe areas are clean enough for overlays when needed
- Contrast supports nearby UI and copy
- Image density matches section importance

### 2. Content Fit

Check whether each image supports a distinct message.

- The image clarifies the page or section idea
- The subject is not merely decorative
- The metaphor is understandable without a caption
- The image does not imply services, claims, or outcomes the brand does not offer
- The image avoids irrelevant props, settings, or symbolic clutter

### 3. Cohesion

Check whether the set shares a recognizable system.

- Shared palette or color grade
- Shared lighting or contrast posture
- Shared material world or graphic language
- Consistent realism level
- Consistent treatment of overlays, texture, grain, or depth
- Consistent representation standards

### 4. Variation

Check whether the images avoid becoming near-duplicates.

- Different image roles are visible at a glance
- Camera distance varies intentionally
- Subject count varies intentionally
- Settings do not repeat without purpose
- Diagram motifs do not repeat as filler
- Human scenes do not repeat the same conversation, meeting, or gaze pattern

### 5. Brand Fit

Check whether the set matches the brand's source of truth.

- Audience sophistication is right
- Tone matches the brand voice
- Colors and materials fit the design system
- Imagery does not contradict do/don't guidance
- The set feels ownable rather than category-generic

### 6. Impact And Value

Check whether the set raises the perceived value of the offer.

- The strongest image feels suitable for the buyer's decision level
- The suite would not feel underpowered beside direct competitors
- At least one image creates a memorable visual moment the brand can own
- Visuals show strategic specificity, not only atmosphere
- The set avoids stock consulting tropes, toy-like diagrams, and decorative data effects

### 7. Artifact Control

Check for generation defects.

- Malformed anatomy, faces, hands, furniture, or geometry
- Accidental readable text, letters, numbers, pseudo-logos, or watermark-like marks
- Incoherent dashboards, charts, or UI fragments
- Over-sharpening, over-blur, plastic skin, or fake depth
- Cropped limbs, duplicate figures, or inconsistent reflections

## Decision Labels

| Label | Meaning | Action |
|-------|---------|--------|
| Keep | Works in placement and strengthens the set | Use or implement |
| Refine | Concept is right, execution needs targeted repair | Rewrite prompt and regenerate |
| Replace | Execution may be polished, but mode or concept is wrong | Re-plan placement |
| Cut | Redundant or unnecessary | Remove from set |

## Regeneration Guidance

When refining, change the cause of the failure rather than adding more adjectives.

- Repetition problem: change visual mode, camera distance, subject count, or metaphor
- Brand drift: restate shared palette, materials, and forbidden cues
- Placement failure: specify crop, text-safe zone, focal placement, and density
- Artifact problem: simplify the subject and reduce small text, hands, interfaces, or crowded geometry
- Generic stock feel: add more specific audience, environment, decision moment, and composition logic
- Weak cohesion: repeat the shared system rules in every prompt, not just the first prompt

## Final Set Questions

- Would the site feel worse if this image were removed?
- Could a competitor use the same image unchanged?
- Does the strongest image make the offer feel more valuable?
- Are two images solving the same job?
- Does any repeated motif carry meaning, or is it decoration?
- Does the set have enough quiet images between high-information images?
- Is there a deliberate balance of human trust, system logic, and brand atmosphere?
