# Campaign Design & Behavior Change

Frameworks for designing campaigns, funnels, and experiences that drive action. Covers behavior models, nudge architecture, and UX psychology.

---

## Hick's Law

Decision time increases with number and complexity of choices. More options → slower decisions → more abandonment.

**Marketing use**: Simplify choices. One clear CTA beats three. Fewer form fields beat more. Every option added to page increases cognitive load and drop-off probability.

---

## AIDA Funnel

Attention → Interest → Desire → Action. Classic journey model — still works.

**Marketing use**: Structure pages and campaigns to move through each stage in order. Capture attention before building desire. Don't ask for action before establishing interest. Match content to stage.

---

## Rule of 7

Prospects need roughly 7 touchpoints before converting. One ad rarely converts alone.

**Marketing use**: Build multi-touch campaigns across channels. Retargeting, email sequences, consistent presence compound. Plan for 7+ interactions, not single-shot conversion.

---

## Nudge Theory / Choice Architecture

Small changes in how choices presented significantly influence decisions. Context shapes choice more than preference does.

**Marketing use**: Default selections, strategic ordering, friction reduction guide behavior without restricting choice. Move desired option to top. Pre-check preferred tier. Remove unnecessary steps.

---

## BJ Fogg Behavior Model

Behavior = Motivation × Ability × Prompt. All three must be present simultaneously for action to occur.

**Marketing use**: High motivation but hard to do = won't happen. Easy to do but no prompt = won't happen. Diagnose which element missing:
- Low motivation → improve value proposition
- Low ability → reduce friction
- No prompt → add clear CTA at right moment

---

## EAST Framework

Make desired behavior: **E**asy, **A**ttractive, **S**ocial, **T**imely. Four levers for behavior change.

**Marketing use**:
- **Easy**: Reduce steps, pre-fill forms, one-click actions
- **Attractive**: Visual appeal, clear benefit, personalization
- **Social**: Show others doing it, peer comparisons, testimonials
- **Timely**: Ask at right moment — after value experienced, before need peaks

---

## COM-B Model

Behavior requires three conditions: **C**apability, **O**pportunity, **M**otivation.

**Marketing use**: Can they do it (capability)? Is path clear (opportunity)? Do they want to (motivation)? Address all three. Tutorial fixes capability. Removing friction fixes opportunity. Case study fixes motivation.

---

## Activation Energy

Initial energy required to start something. High activation energy prevents action even if task easy overall.

**Marketing use**: Reduce starting friction. Pre-fill forms, offer templates, show quick wins. Make first step trivially easy. "Get started in 30 seconds" removes activation barrier.

---

## North Star Metric

One metric that best captures value delivered to customers. Single focus creates alignment.

**Marketing use**: Identify North Star (active users, completed projects, revenue per customer). Align all campaign efforts toward moving this metric. Prevents optimizing vanity metrics that don't matter.

---

## Cobra Effect

When incentives backfire and produce opposite of intended results. Good intentions, bad design.

**Marketing use**: Test incentive structures before scaling. Referral bonus might attract low-quality referrals gaming system. Discount code might cannibalize full-price sales. Model second-order effects of incentives.

---

## Designing for Behavior Change — Integration Pattern

Combine frameworks for campaign design:

### Diagnosis Phase
1. Use **BJ Fogg** to identify which element missing (motivation, ability, prompt)
2. Use **COM-B** to verify capability, opportunity, motivation present
3. Map where in **AIDA** funnel problem occurs

### Design Phase
1. Apply **EAST** framework to solution design
2. Use **Hick's Law** to simplify choices
3. Lower **Activation Energy** for first action
4. Add **Nudge** elements (defaults, ordering, friction removal)

### Measurement Phase
1. Track **North Star Metric** as primary outcome
2. Monitor for **Cobra Effect** — unintended consequences
3. Plan **Rule of 7** touchpoints across journey
