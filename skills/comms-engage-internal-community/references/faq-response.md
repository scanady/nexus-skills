# FAQ Response Template

Use when internal stakeholders repeatedly ask similar policy or process questions.

## Required Inputs

- Audience and context
- Question list
- Source-of-truth policy references
- Exceptions and escalation path

## Structure

1. Intro context
2. Q&A pairs
3. Exceptions and edge cases
4. Escalation contact / next step

## Writing Rules

- Keep each answer direct and specific.
- Reference policy source when available.
- Avoid legalistic wording unless explicitly required.
