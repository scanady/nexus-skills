# Content Structure Frameworks

Reference for classifying source content into structural archetypes. Load this when analyzing unfamiliar formats or when the content doesn't fit a standard category.

## Format Archetypes

### Narrative Formats
| Archetype | Structure | Common In |
|-----------|-----------|-----------|
| **Inverted pyramid** | Lead → supporting details → background | News, press releases, announcements |
| **Problem-solution** | Pain → agitation → resolution | Blog posts, case studies, sales pages |
| **Hero's journey** | Status quo → challenge → transformation → new normal | Brand stories, testimonials, about pages |
| **Before-after-bridge** | Old state → new state → how to get there | Marketing copy, product launches |
| **Narrative arc** | Setup → rising tension → climax → resolution | Long-form articles, essays, newsletters |

### Structured Formats
| Archetype | Structure | Common In |
|-----------|-----------|-----------|
| **Listicle** | Numbered or bulleted items with optional framing | Blog posts, social content, roundups |
| **How-to / Tutorial** | Goal → prerequisites → sequential steps → outcome | Documentation, educational content |
| **Q&A / FAQ** | Question-answer pairs, grouped or sequential | Support content, landing pages |
| **Comparison** | Criteria → option A vs. option B → recommendation | Reviews, buying guides, decision docs |
| **Framework / Model** | Concept → components → relationships → application | Thought leadership, educational content |

### Persuasive Formats
| Archetype | Structure | Common In |
|-----------|-----------|-----------|
| **AIDA** | Attention → Interest → Desire → Action | Ads, landing pages, email campaigns |
| **PAS** | Problem → Agitation → Solution | Sales copy, outreach emails |
| **Feature-benefit** | Capability → what it means for you | Product pages, pitch decks |
| **Social proof cascade** | Claim → evidence → testimonial → CTA | Landing pages, case studies |
| **Contrarian / hot take** | Conventional wisdom → why it's wrong → better approach | Thought leadership, social posts |

### Conversational Formats
| Archetype | Structure | Common In |
|-----------|-----------|-----------|
| **Stream of consciousness** | Associative flow, loose structure, personal voice | Newsletters, personal blogs, social |
| **Letter / memo** | Salutation → context → body → ask/close | Email campaigns, internal comms |
| **Thread / carousel** | Connected micro-units, each self-contained but sequential | Social media (X threads, LinkedIn carousels) |
| **Dialogue / interview** | Back-and-forth exchange, real or constructed | Podcasts, interview articles |

## Tone Spectrum Reference

Use these spectrums to position the source content's tone precisely rather than using vague labels.

### Formality Axis
`Academic → Professional → Business casual → Conversational → Casual → Irreverent`

### Emotional Axis
`Detached → Measured → Warm → Passionate → Urgent → Intense`

### Authority Axis
`Peer → Guide → Mentor → Expert → Authority → Institution`

### Energy Axis
`Contemplative → Steady → Energetic → Punchy → High-octane → Frenetic`

## Hook Type Taxonomy

When classifying opening patterns:

| Hook Type | Mechanism | Example Pattern |
|-----------|-----------|-----------------|
| **Statistical** | Surprising number that reframes | "X% of [group] [unexpected fact]" |
| **Question** | Provokes curiosity or self-reflection | Direct or rhetorical question |
| **Story / Anecdote** | Narrative immersion | Personal or third-party micro-story |
| **Contrarian** | Challenges assumption | "Everything you know about X is wrong" |
| **Declarative** | Bold, confident statement | Strong opinion stated as fact |
| **Scene-setting** | Vivid image or scenario | Descriptive, sensory language |
| **Problem statement** | Names a pain the reader feels | "You've tried X and it didn't work" |
| **Promise** | Commits to delivering value | "By the end of this, you'll know how to..." |

## Closing Pattern Taxonomy

When classifying ending patterns:

| Close Type | Mechanism |
|------------|-----------|
| **Call to action** | Direct request for next step |
| **Summary** | Recap of key points |
| **Callback** | Returns to the opening hook or theme |
| **Open question** | Leaves reader thinking |
| **Vision / future** | Paints a picture of what's possible |
| **Mic drop** | Single powerful closing line |
| **Resource list** | Links or next steps for further learning |
