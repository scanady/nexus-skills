# Change Monitoring and Governance

Use this model to keep compliance intelligence current and auditable.

## Monitoring Cadence

| Cadence | Purpose | Typical outputs |
|---|---|---|
| Weekly | Detect urgent shifts and enforcement signals | Rapid alerts, watchlist updates |
| Monthly | Consolidate meaningful updates | Standards register delta report |
| Quarterly | Validate control framework health | Governance review and cleanup plan |

## Trigger-Based Updates

Run unscheduled updates when:

- Material enforcement action affects relevant claims/channels
- New state bulletin materially changes marketing expectations
- Company policy changes claim approval requirements
- Product changes alter suitability or disclosure constraints

## Governance Checklist

- Confirm source dates and supersession status
- Validate owner assignment for each active control
- Close stale open questions or re-escalate
- Document interpretation decisions in a decision log
- Publish downstream impact summary for review teams
